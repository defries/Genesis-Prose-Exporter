== Description ==

This plugin hooks Prose settings into Genesis Exporter, allowing Prose Design Settings to be exported.

== Installation ==

Installation of this plugin works like any other plugin out there. Either:

1. Upload the zip file to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the export function of the Genesis Framework which will now include the Prose settings

Or search for the plugin via your plugins menu.
